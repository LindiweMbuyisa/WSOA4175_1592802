Creative Projects

Car Crash

Car Crash is a 2D single player game. The player controls a car which is on the wrong side of the road
The goal of the game is to avoid the incoming cars. In the game there is powerups that can make the car invisible making it easier to dodge the cars, powerups that make the car small and retract the car back to its original size.


Illusion

Illusion is a 2D picture sliding block puzzle. The player has to rearrange the blocks by selecting/clicking the block to move up,down, left or right. This game which a recreation of the classic puzzle sliding block puzzle game, it takes a different approach in terms of the rotating puzzle blocks and the pictures in the game which play around illusion.



Find

This game is about finding the right path to ensure each of the block characters reach their goal however the player is in control of both the characters. This game features inverse mechanic- whereby the block characters move in opposite direction. So while one of the characters moves up, the other moves down.



BLM

This is a motion graphics project, which I wanted to explore typography, but this project also serves as my response to the black lives matter movement which is an important movement.



FUTUREISINTHEAIR
